# Original Files

This directory is a read-only snapshot of the Claude/static prototype before the React app scaffold was introduced.

Use it as the visual and behavior reference while porting pages into React:

- `html/` contains the original root pages.
- `css/` contains the original global styles and design tokens.
- `js/` contains the original vanilla page helpers.
- `src-canvas/` contains the original Babel/browser-global canvas implementation.
- `data/` contains the original sample graph data.

Do not edit files in this directory during normal implementation. Refresh the snapshot only when explicitly requested.
